## 关键事件与解决

1. **用户实测抓出两个烟测漏掉的 bug**：① 全部弹层黑边框+错误阴影——根因是手动装 shadcn 漏了 `@layer base { * { border-color: var(--border) } }`（Tailwind v4 preflight bare border=currentColor）；② 显示比例小档底部留白/大档整页滚动——CSS zoom 不缩 vh，修法 `--zoom-inverse` 三档 + `.h-app { height: calc(100vh*var(--zoom-inverse,1)) }`，9 组合（3 zoom×3 布局）实测 gap=0 scroll=0。
2. **弹层/动效已逐字对齐原型**（commit 7b9f26a + 6371805）：modal-in .18s、sheet-in-right .28s cubic-bezier(.4,0,.2,1)、遮罩 rgba(0,0,0,.22)+blur(6px)、弹窗 shadow 0 12px 40px .16 + rounded-14、子菜单 grid 0fr↔1fr .26s、毛玻璃 blur(14px) saturate(1.6)（已实滚截图验证）、tooltip 恒 #1f2329（--tooltip-bg token）、圆角全档 --radius-4…14 乘法 token（tailwind-merge 需 extendTailwindMerge 注册数字档冲突组，已根治）。
3. **多 agent 协调事故（已化解）**：批次 B 的 implementer 上下文过重后行为退化（静默 idle、消息交叉把用户 bug 当"已知偏差"），换人接手时发生短暂同工作区并行编辑——幸而改动互补，最终由接手者统一验证 commit。教训：同工作区只允许一个 implementer 活跃；裁决消息要逐项编号+标注【已裁决】。
4. **进程纪律**：subagent 曾 `pkill -f vite` 误杀用户其他项目的 dev server——已立规矩：只按端口精准 kill（`lsof -ti :5173 | xargs kill`）。
